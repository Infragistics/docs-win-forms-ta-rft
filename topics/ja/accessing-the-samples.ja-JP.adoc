= サンプルへのアクセス

{ProductName} (Rational Functional Tester) のサンプル スクリプトは、Samples フォルダーにインストールされます。

*Windows 10 のディレクトリ*

C:\Users\Public\Documents\Infragistics\20{ProductVersion}\WinForms Test Automation (IBM)\Samples

*古い OS を使用している場合、サンプルはルートの「Users」フォルダーから始まる同様の場所にあります。* 

サンプル スクリプトは、Java と VB.NET のバージョンがあり、いずれも同じサンプル アプリケーションに対して実行できます。

各スクリプトは、実行前に同じアプリケーションを自動的にロードするように設定されます。

製品の Start Menu のショートカットからサンプル スクリプトを開き、実行することができます。
