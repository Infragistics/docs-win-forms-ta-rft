= サポートされるデータ確認ポイント

データ確認ポイントを使用して、テスト済みアプリケーションのコントロール データをテストできます。確認ポイントを作成すると、確認済みデータのスナップショットが保存されます。次に確認ポイントの再生セッション中に、保存されたスナップショットと比較してコントロールのデータに変更がないことを確認します。この方法でアプリケーションの潜在的なエラーを発見することができます。

データ確認ポイントを使用する場合は、リテラル値とテスト データ プールから取得したデータ駆動型値の両方を使用できます。

[NOTE]
====
*注:* IBM Rational Functional Tester による確認ポイントの使用方法の詳細は、Rational Functional Tester と一緒にインストールされる IBM Rational Software Development Platform ヘルプ (ユーザー ガイド > 記録 > 確認ポイント) を参照してください。
====
