= WinToolbarsManager の既知の問題

このトピックは、WinToolbarsManager™ コントロールの既知の問題について説明します。

* Rational Functional Tester iFix001 をインストールした場合、 WinToolbarsManager の確認ポイントは追加できません。ただし、既に作成済みの確認ポイントは問題なく再生できます。
* Windows Vista で記録中にツールバーまたはツール確認ポイントを追加することはできません。
