= サポートされるコントロールおよびコンポーネント

{ProductName} (Rational Functional Tester) は、対応する Ultimate UI for Windows Forms リリース内のすべての視覚的な Windows Forms コントロールおよびコンポーネントをサポートします。一部の Ultimate UI for Windows Forms コントロールおよびコンポーネントは、非視覚要素またはユーザー インタラクションが不要なため、対応するカスタム ライブラリを必要としない、またはライブラリ自体がありません。ただし、これらのコントロールおよびコンポーネントは、{ProductName} (Rational Functional Tester) で引き続きサポートされます。

{ProductName} (Rational Functional Tester) は、以下の視覚化 Ultimate UI for Windows Forms コントロールをサポートします。

コントロールおよびコンポーネントは、CLR4 およびバージョンフリーのアセンブリで利用できます。
