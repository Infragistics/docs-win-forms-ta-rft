= 2012 Volume 1 の新機能

*概要*

{ProductName} (Rational Functional Tester) 2012 volume 1 リリースでは、Ultimate UI for Windows Forms 2012 Volume 1 コントロールのスイート全体のテスト機能をサポートしています。

*WinGrid - 空メッセージの確認ポイント*

新しい*_空メッセージ_* の確認ポイントが追加されました。これによりテスターは、グリッドの空メッセージ機能設定に基づき、グリッド内に表示される異なるタイプの空メッセージを確認できるようになります。新しい確認ポイントは、以下の情報が含まれています。

* MessageType -  _{NoDataSourceMessage, NoRowsInDataSourceMessage, NoVisibleRowsMessage}_  各タイプについて
* MessageEnabled
* MessageText
