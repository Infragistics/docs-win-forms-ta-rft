= Supported Controls and Components

{ProductName} (Rational Functional Tester) contains support for all visual Windows Forms controls and components in the corresponding Ultimate UI for Windows Forms release. Some Ultimate UI for Windows Forms controls and components are either non-visual elements or require no user interaction and, therefore, do not require or have corresponding custom libraries. However, these controls and components are still supported by {ProductName} (Rational Functional Tester).

{ProductName} (Rational Functional Tester) supports the following visual Ultimate UI for Windows Forms controls.

The controls and components are available with CLR4 and version-free assemblies.
