= Accessing the Samples

{ProductName} (Rational Functional Tester) sample scripts are installed in the Samples folder.

*Windows 10 Location*

C:\Users\Public\Documents\Infragistics\20{ProductVersion}\WinForms Test Automation (IBM)\Samples

*Samples will be found in a similar location starting at the root "Users" folder when using an older OS.* 

There is a Java and VB.NET version of the sample script. both run against the same sample application.

Each script is set up to automatically load the sample application prior to running.

You can open and run the sample script from the Start Menu shortcuts of the product.
